# Macaron Pink 🍰

A sweet **pastel light theme** inspired by French macarons.

## Features

- Cream background (`#FFF8EC`)
- Burgundy text
- Accents in *pink*, *lavender*, *mint*, *lemon*

## Installation

```bash
code --install-extension macaron-pink-0.0.1.vsix
```
