# Macaron Pink

A sweet pastel **light** theme for Cursor / VS Code, inspired by French macarons.

## Palette

| Role | Color | |
|---|---|---|
| Background | `#fff9ec` | cream |
| Foreground | `#5d2a42` | burgundy |
| Accent | `#fb6376` | watermelon pink |
| Keyword | `#d5467a` | deep pink |
| String | `#3a9b6e` | mint |
| Function | `#9b5bd2` | lavender |
| Property | `#2d8ab5` | sky blue |
| Number | `#b38b00` | honey |
| Comment | `#b88ca5` | muted mauve |

## Preview

1. Open this folder in Cursor
2. Press `F5` to launch an Extension Development Host window
3. In the new window: `Cmd+Shift+P` → `Preferences: Color Theme` → **Macaron Pink**
